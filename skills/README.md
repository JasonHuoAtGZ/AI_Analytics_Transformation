# Skill Registry

This folder contains reusable skills, each in its own subfolder with a `SKILL.md` file.

| Skill | Folder | Status | Description |
|---|---|---|---|
| NL-to-SQL | `nl-to-sql/` | 🔴 Not created | Convert natural language business questions into DuckDB SQL |
| Data Generation | `data-generation/` | 🔴 Not created | Generate realistic life insurance dummy data |
| DuckDB Query Patterns | `duckdb-query-patterns/` | 🔴 Not created | Common DuckDB query templates for customer analytics |
| Streamlit Patterns | `streamlit-patterns/` | 🔴 Not created | Streamlit UI conventions for the agent interface |

## When to create a new skill

Create a new skill when:
- A capability is reusable across multiple jobs
- A pattern or workflow needs to be standardized
- An agent repeatedly needs the same set of instructions
