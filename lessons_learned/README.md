# Lessons Learned

Chronological log of every mistake, error, blocker, and resolution encountered during the project.

## Entry Format

Each lesson is a dated Markdown file with:
- **Date**: when it happened
- **Context**: what we were trying to do
- **Error**: exact error message or behavior observed
- **Root cause**: why it happened
- **Resolution**: what fixed it
- **Prevention**: how to avoid it going forward
